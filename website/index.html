核心能力

不是泛泛的 RPA 演示，而是围绕 Office 真实操作路径打磨。

🎬 一次录制

记录点击、输入、快捷键、滚轮与拖选。自动合并连续按键，减少冗余步骤。

🧱 模块积木

把「复制→打开网页→粘贴」存成模块，插入到任意流程间隙，避免重复录制。

🔁 表格循环

Excel 拖选行数或 WPS 输入行数，录一遍循环体，自动重复 N 行并下移。

🌐 浏览器协同

针对 Gmail、内部系统的粘贴与等待加载优化，减少焦点跑偏。

🖥️ Web 控制台

可选本地网页 UI 管理流程；配合隧道可远程触发（执行仍在办公电脑上）。

🔒 本机执行

无需把账号密码交给云端机器人。流程 JSON 存在本机，适合内网环境。

三步上手

  1. 1

    安装并授权

    Mac / Windows 安装后，在系统设置中开启「辅助功能」与「输入监控」。

  2. 2

    录制流程

    切到 Excel 或 WPS，点录制，做一遍日常操作；需要批量时插入循环标记。

  3. 3

    保存与回放

    命名保存，之后一键 Run。可加速回放、插入文字步骤、自动纠错。

常见问题

这是 SaaS 吗？数据会上传吗？

不是。流程与录屏截图默认保存在本机用户目录。可选 Web 控制台仅用于管理，不要求公有云。

能控制我电脑上的 WPS / 微信吗？

能控制已安装桌面应用中的标准 UI 操作（点击、键盘）。部分安全软件或远程桌面可能限制模拟输入。

Mac 和 Windows 功能一样吗？

核心录制回放一致。Excel 拖选循环在 macOS 上最完整；Windows 部分场景建议用行数输入。
