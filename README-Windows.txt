OfficeLego - Windows Quick Start
================================

INSTALL (no Python needed)
--------------------------
1. Extract the FULL zip (you must get a "OfficeLego" folder)
2. Open the OfficeLego folder — you should see:
   - OfficeLego.exe
   - _internal  (folder — required, do not delete)
   - START.bat  (recommended — double-click this)
   - windows_launch.bat
   - README-Windows.txt
3. Double-click START.bat  OR  OfficeLego.exe
4. If SmartScreen blocks: click "More info" -> "Run anyway"
5. If antivirus asks: choose Allow / 允许

IMPORTANT
-----------
- Do NOT copy only OfficeLego.exe to Desktop — it will NOT work alone
- Keep OfficeLego.exe and _internal folder together

IF IT STILL FAILS
-----------------
Open this file and send it to support:
  C:\Users\YOUR_NAME\AppData\Local\OfficeLego\officelego-error.log

Recording tips:
- Record clicks on YOUR own PC (coordinates differ per machine)
- Browser zoom: keep 100%
- Ctrl+C / Ctrl+V = copy / paste
