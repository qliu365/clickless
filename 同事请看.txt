Clickless 自动化助手 — Windows 使用说明
========================================

【安装】无需 Python
----------------------------------------
1. 解压 Clickless-win.zip（必须得到整个 Clickless 文件夹）
2. 打开 Clickless 文件夹，应看到：
   - Clickless.exe
   - _internal 文件夹（必须有，不能删）
   - windows_launch.bat
3. 双击 windows_launch.bat 或 Clickless.exe
4. Windows 安全提示：点「更多信息」→「仍要运行」
5. 杀毒软件拦截：选「允许」


【常见错误：打不开 / 闪退】
----------------------------------------
❌ 只把 Clickless.exe 复制到桌面 → 不能用
   必须保留 exe 和 _internal 文件夹在一起

❌ 只解压了 exe 没解压 _internal → 不能用

✅ 正确：整个 Clickless 文件夹一起拷贝


【若仍失败】
----------------------------------------
查看错误日志（发给技术支持）：
  C:\Users\你的用户名\AppData\Local\Clickless\clickless-error.log


【怎么用】
----------------------------------------
录制：点红按钮 → 切到浏览器/Excel → 右下角「停止」
运行：点「运行选中」→ 切到目标窗口 → 等 5 秒

支持：点击、双击、拖拽、滚轮、Ctrl+C/V 复制粘贴


【注意】
----------------------------------------
- 点击坐标要在自己电脑上录
- 浏览器缩放建议 100%
- 位置不准：运行时倒计时内点一下应对准的位置

有问题联系发你文件的人。
