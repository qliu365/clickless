Clickless 自动化助手 — Windows 使用说明
========================================

【安装】无需 Python
----------------------------------------
1. 解压 Clickless-win.zip
2. 进入 Clickless 文件夹
3. 双击 Clickless.exe
4. 杀毒软件拦截时选「允许」


【怎么用】
----------------------------------------
录制：点红按钮 → 切到浏览器操作 → 右下角「停止」
运行：点绿色「运行」→ 切到浏览器 → 点一下网页空白处 → 等 5 秒

注意：
- 点击坐标要在自己电脑上录，不能直接用别人的
- 顺序：输入 → 点击 → 输入 → 点击

有问题联系发你文件的人。
